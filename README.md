# Product Store App

App for using login and file storage using Firebase.

- Using Bloc
- Forms using Bloc
- Login UI design 
- InheritedWidget
- Connect inputs with Streams
- Validations.
- CRUD to firebase 
- Calling REST services 
- Image loading
- Camera Access 
- Camera Gallery Access
- New users on firebase
- Firebase Auth Rest API
- Tokens Handling
